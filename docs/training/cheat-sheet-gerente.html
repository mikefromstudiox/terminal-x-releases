1. Tarjeta de Autorización de Gerente

Tu tarjeta tiene un código Code128 impreso. La cajera la pasa por el scanner cuando el sistema bloquea una operación sensible.

¿Qué autoriza tu tarjeta?

  • Anular (void) un ticket
  • Emitir nota de crédito / devolución
  • Descuentos > RD$500 o > 15%
  • Cambios de precio en línea del carrito
  • Ajustes manuales de inventario
  • Retiros de caja chica

PIN de respaldo

Si pierdes la tarjeta o no la tienes a mano, usa tu PIN de gerente. Cada autorización queda registrada con tu nombre en el log de actividad.

Responsabilidad: cada autorización queda amarrada a tu usuario. Si autorizas descuentos falsos o voids injustificados, el dueño lo ve todo en el dashboard.

2. Aperturar turno POR el cajero

Desde v2.13.2 solo la cajera puede abrir su propio turno. Pero si eres owner / manager / CFO / contador, tú puedes abrirlo por ella:

  1. Entra a Cuadre.
  2. Elige la cajera en el selector.
  3. Escribe el monto inicial RD$ de la gaveta.
  4. Confirma — la cajera ya puede empezar a vender con su PIN.

3. Cerrar turno / Hacer cuadre

  1. Entra a Cuadre → selecciona el turno abierto.
  2. Cuenta el efectivo físico con la cajera presente.
  3. Entra el monto contado — el sistema muestra la diferencia.
  4. Si hay discrepancia > RD$50, el sistema lo registra automáticamente en el log de actividad con severidad alta.
  5. Imprime el cuadre, firmen ambos, archiva.

5. Caja Chica

  • Todo retiro requiere tu autorización (tarjeta o PIN).
  • Cada retiro debe tener concepto y, si es posible, comprobante físico grapado.
  • Reintegros al final del mes o cuando se agote el fondo.
  • Todo queda en el log de actividad.

6. Inventario — Conteo y Ajustes

Conteo físico

  1. Inventario → Conteo Físico → iniciar sesión.
  2. Imprime hoja en blanco, entrega al equipo para contar.
  3. Entra los conteos → sistema genera reporte de varianza con pérdidas en RD$.
  4. Exporta PDF y CSV para el dueño.

Ajustes manuales

Requieren autorización tuya y un motivo (rotura, vencimiento, ajuste administrativo). Sin motivo, el sistema rechaza.

7. Reportes de Productos (Pro PLUS+)

  • Unidades vendidas, ingreso, costo, utilidad por SKU.
  • El margen se muestra ex-ITBIS — el sistema divide el precio por 1.18 cuando el producto aplica ITBIS. No te asustes si el margen se ve más bajo que antes, es el correcto.
  • Filtra por categoría, subtipo, rango de fechas.
  • Exporta a PDF o CSV para el dueño / contador.

8. Escalamiento al dueño

Escala al dueño (o a soporte Terminal X vía WhatsApp) si:
  • Discrepancia de cuadre repetida con la misma cajera
  • Sospecha de fraude o robo
  • Error del sistema que no puedes resolver (rechazos e-CF persistentes, impresora muerta, sync caído)
  • Empleado pide cambio de PIN / tarjeta
  • Certificado e-CF próximo a expirar (el sistema avisa en el dashboard)
