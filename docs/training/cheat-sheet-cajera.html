1. Apertura de Turno

  • Solo la cajera abre su propio turno. El dueño, gerente, CFO o contador no abre turno — ellos lo abren por la cajera desde Cuadre.
  • Al entrar al POS, el sistema pide monto inicial en RD$ (el efectivo con que arrancas el día).
  • Si el navegador sugiere contraseñas (ícono de llave 1Password/LastPass), ignóralo — el campo está bloqueado para autofill.
  • Confirma y ya estás lista para vender.

2. Cobrar — Métodos de pago

MétodoQué hacer
EfectivoEscribe monto recibido → sistema muestra el vuelto → gaveta se abre sola.
TarjetaPasa tarjeta en POS externo → anota últimos 4 dígitos si el sistema los pide.
TransferenciaMuestra número/QR al cliente → confirma llegada antes de imprimir.
MixtoDividir total entre varios métodos. El botón Agregar pago permite combinar.
Crédito (fiado)Solo si el cliente está registrado y tiene cupo. Nunca a cliente nuevo sin aprobación.

3. e-CF con RNC — Cliente pide factura

  • Cuando el cliente pida factura, presiona el botón RNC en el modal de cobro.
  • Escribe el RNC (9 dígitos) o la cédula (11 dígitos) — el sistema busca automáticamente el nombre en la base DGII.
  • Si no aparece, pídele al cliente el nombre exacto y dirección; sistema lo guarda para próximas compras.
  • Sistema emite el tipo de e-CF correcto (E31, E32, E34, etc.) automáticamente — tú no eliges.
Si DGII no responde (internet caído), el sistema guarda el e-CF en cola y lo envía después. Sigues vendiendo normal.

4. Devoluciones y Notas de Crédito

Requiere autorización de gerente. El sistema pedirá la Tarjeta de Autorización o PIN del manager. Nunca proceses una devolución tú sola.
  1. Busca el ticket original (por número o fecha).
  2. Marca los ítems que se devuelven.
  3. Llama al gerente → pasa su tarjeta o digita PIN.
  4. Devuelve el efectivo o genera crédito a cuenta.

5. Descuentos

  • Descuentos < RD$500 y < 15%: los haces tú.
  • Descuentos > RD$500 o > 15%: el sistema bloquea y pide autorización de gerente.
  • Siempre pon un motivo claro (ej: "cliente frecuente", "producto con golpe").

6. Kiosco bloqueado — Desbloquear

  • Si dejas la caja sin actividad por varios minutos, el sistema se bloquea automáticamente (modo kiosco).
  • Entra tu PIN para desbloquear — tu sesión se preserva, no pierdes la venta en curso.
  • Si otra cajera quiere usar la caja, debe cerrar tu sesión primero.

7. Cierre de Turno — Cuadre

  1. Cuenta el efectivo físico en la gaveta.
  2. Entra a Cuadre → escribe el monto contado.
  3. Sistema compara con el esperado y muestra diferencia.
  4. Si la discrepancia es > RD$50, reporta al gerente antes de cerrar.
  5. Imprime el cuadre y entrégalo al encargado del cierre.

8. Problemas de impresora

  • Si no imprime: revisa el cable USB, el papel y que esté encendida.
  • Presiona Reintentar en el banner rojo — casi siempre funciona.
  • Si el banner rojo persiste, llama al gerente o a soporte: +1 (809) 828-2971.

9. Lo que NUNCA debes hacer

  • Nunca anules un ticket sin aprobación del gerente (es fraude contra el negocio y DGII).
  • Nunca cierres sesión sin hacer cuadre.
  • Nunca compartas tu PIN con otra cajera ni con el cliente.
  • Nunca regales producto ni apliques precios distintos sin registrarlos en el sistema.
  • Nunca borres una venta del sistema — aunque haya error, emite nota de crédito.
